A simple command line pong game made in rust! 
The only external library it uses is crossterm (for detecting keypress), everything else is made from scratch.
NO AI or LLMs was used for makeing this. Everything is made by hand. (with arch linux & neovim btw)
Made for fun!

## Prerequisites
* Ensure rust is installed
* Ensure git is installed
  
## Running
```
git clone https://github.com/RayAntariksha/Pong.git
cd Pong
cargo run
```
