A simple command line pong game made in rust! <br>
The only external library it uses is crossterm (for detecting keypress), everything else is made from scratch.<br>
NO AI or LLMs was used for making this. Everything is made by hand.<br>
Made for fun!

## Prerequisites
* Ensure rust is installed
* Ensure git is installed
  
## Running
```
git clone https://github.com/RayAntariksha/Pong.git
cd Pong
cargo run
```
